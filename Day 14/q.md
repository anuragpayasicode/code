1. Student Grade System (Very Common)

Problem Statement

Calculate grade based on marks.
What you have to do
Create class Student
Properties: name, marks
Method:
getGrade()

Rules
90+ → A

75–89 → B

50–74 → C

<50 → Fail

Input

Aman = 88
Neha = 45


Expected Output


Aman Grade: B
Neha Grade: Fail
